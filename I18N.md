# English / Russian interface

Language buttons are in the top header.

## What is translated

- static site copy;
- track display titles and koans;
- the lower terminal/signal band;
- Passenger Audit questions and answers;
- route warnings and support copy;
- progress/counter status.

## What is intentionally not translated

- matrix glyphs;
- QR word-field fragments;
- puzzle keys;
- GitHub routes;
- hashes and proof codes;
- wallet addresses;
- raw code/source identifiers.

## Files

```text
js/i18n.js           static copy and display translations
data/oracle.en.enc   English encrypted oracle
data/oracle.ru.enc   Russian encrypted oracle
data/oracle.enc      English compatibility alias
```

The selected language is stored locally as `lsh_lang`. Russian is selected automatically for a Russian browser only if the visitor has not already chosen a language.

After changing `js/i18n.js`, run:

```bash
node --check js/i18n.js
```

The encrypted oracle format is documented in `ORACLE.md`.
